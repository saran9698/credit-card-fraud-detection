# Credit Card Fraud Detection

## Project Overview
This project detects fraudulent credit card transactions using machine learning techniques.

## Objective
To identify suspicious transactions and reduce financial fraud.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data preprocessing
- Fraud transaction analysis
- Data visualization
- Machine learning prediction

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Dataset
Credit Card Fraud Detection Dataset from Kaggle.

## Results
The model successfully identifies fraudulent transactions with high accuracy.
